**DESIGN SYSTEM (REQUIRED):**
- Platform: Web, Desktop-first
- Theme: Dark, professional, authoritative, high-contrast
- Background: Deep Navy/Black (#050505)
- Surface: Dark Surface (#121212)
- Primary Accent: Vibrant Cyan-Blue (#00b4d8) for main CTA buttons and glowing highlights
- Secondary Accent: Muted Teal (#2CBFBF) for badges and secondary highlights
- Premium Accent: Deep Purple/Lilac (#6D28D9) for premium packages
- Text Primary: Clean White (#ffffff)
- Typography: Playfair Display for elegant headings, Montserrat for bold high-impact body and UI elements.
- Components: Gently rounded cards (12-16px) with subtle borders that illuminate on hover. Buttons have glowing drop shadows. Use ambient background blur circles for lighting.
